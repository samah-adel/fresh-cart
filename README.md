# 🛒 Fresh Cart

A modern and responsive e-commerce web application built with **React.js**. Users can browse products, manage their shopping cart and wishlist, search for products, and authenticate securely.

## ✨ Features

* 🔐 User Authentication (Register, Login, Forgot Password, Reset Password)
* 🛍️ Browse Products
* 🔍 Search Products
* ❤️ Add / Remove Wishlist Items
* 🛒 Shopping Cart Management
* ➕ Increase & Decrease Product Quantity
* 🗑️ Remove Products & Clear Cart
* 📦 Product Details Page
* 📂 Categories & Brands
* 📱 Fully Responsive Design
* ⏳ Loading States
* ❌ Error Handling
* 🔔 Toast Notifications

---

## 🛠️ Built With

* React.js
* React Router DOM
* React Context API
* React Query (TanStack Query)
* Axios
* Tailwind CSS
* Swiper.js
* React Hot Toast
* React Icons
